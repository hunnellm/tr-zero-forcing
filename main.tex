\begin{frame}{Standard Zero Forcing: Core Idea}
\small
Let $G=(V,E)$ be a simple graph.
\begin{itemize}
  \item Each vertex is initially colored either blue (active) or white (inactive).
  \item Start with an initial active set $B_0\subseteq V$.
  \item Apply the color-change rule repeatedly.
\end{itemize}

\vspace{0.2em}
\textbf{Color-change rule:} If a blue vertex $u$ has exactly one white neighbor $v$, then $u$ \emph{forces} $v$ to become blue. We write $u\to v$.
\end{frame}

\begin{frame}{Process and Terminology}
\small
\begin{itemize}
  \item A sequence of valid forces is a \emph{forcing process}.
  \item If every vertex eventually becomes blue, then $B_0$ is a \emph{zero forcing set}.
  \item The minimum size of such a set is the \emph{zero forcing number}:
\end{itemize}

\[
Z(G)=\min\{\abs{B}: B\subseteq V(G),\ B \text{ is a zero forcing set}\}.
\]

\vspace{0.2em}
\textbf{Goal:} choose a small initial active set that propagates to all of $V(G)$.
\end{frame}

\begin{frame}{Takeaway and Transition}
\small
\begin{itemize}
  \item Standard zero forcing tracks deterministic propagation via the unique-white-neighbor rule.
  \item The central optimization problem is minimizing $\abs{B_0}$, giving $Z(G)$.
  \item This classical process is our baseline for $(k,r)$ zero forcing, where force dynamics are generalized.
\end{itemize}

\vspace{0.35em}
\centering
\large Next: extending from $Z(G)$ to broadcast zero forcing.
\end{frame}

\begin{frame}
  \frametitle{$(k,r)$ zero forcing}
\begin{itemize}
  \item In some applications, the forcing process corresponds to solving a system of linear equations.
  \item In others, it models the spread of information or influence through a network.
  \item Information becomes outdated or irrelevant over time, so we want to model the decay of influence.
  \item Broadcast zero forcing generalizes the standard process by allowing a vertex to force multiple neighbors at once, subject to a threshold condition.
  \item This leads to the $(k,r)$ zero forcing framework, where $k$ is the threshold and $r$ is the radius of influence.
\end{itemize}

\end{frame}
